# 아싸도우미 웹서비스 

혼자서 학교를 다녀야하는 다양한 상황이 존재합니다. 이에 `혼자하는 학교 생활`에 편의를 제공하고자 본 서비스를 고안하였습니다. 


## 핵심 기능  
- 혼자 다니는 사람을 중심으로 두었기 때문에 혼자 다닐 때 필요한 정보를 다른 데서 일일이 찾아볼 필요 없음
- 글을 작성할 수 있기 때문에 혼자 다니는 사람끼리 정보를 공유할 수 있음
- 익명성이 보장되어 누구나 쉽게 글을 올릴 수 있음
- 필터링 기능을 통해 자신이 원하는 정보를 쉽고 빠르게 얻을 수 있음
